Github basics ✎

Get an account ✎

Our example account is the following:

Github name: greta-p
E-mail: greta-p@duckietown.com

Create a Github account.

Verify email.

Go to settings.

Create an SSH Key on your machine ✎

Generate the ssh key with the command:

$ ssh-keygen -h

The output will look something like this:
